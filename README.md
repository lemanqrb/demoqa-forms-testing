# 🧪 DemoQA Manual Testing

Welcome to my **DemoQA Manual Testing** repository! 🚀

This repository contains QA documentation created while testing different modules of the **DemoQA** website. It demonstrates the complete manual testing process, from planning and designing test cases to reporting bugs and summarizing test execution results.

## 📂 Contents

* 📋 Test Plans
* ✅ Test Cases
* 🐞 Bug Reports
* 📊 Test Summary Reports

## 🧩 Tested Modules

The repository includes testing for various DemoQA modules, such as:

* Forms
* Login / Authentication
* Elements
* Alerts, Frames & Windows
* Widgets
* Interactions
* Book Store Application

*Additional modules may be added over time.*

The purpose of this repository is to document the complete manual testing process for the DemoQA website by creating structured QA documentation, identifying defects, and demonstrating industry-standard software testing practices.
